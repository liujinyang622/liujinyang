# Project Title

This project is a social media application that allows users to create accounts, manage friends, chat, and share posts. It is built using React for the frontend and Node.js with MongoDB for the backend.

## Features
- User Registration and Login
- Friend Management
- Real-time Chat
- Social Feed (Posts with text, images, comments, and likes)

## Getting Started

### Prerequisites
- Node.js
- MongoDB

### Installation
1. Clone the repository
2. Install dependencies for both client and server
   ```bash
   cd client
   npm install
   
   cd ../server
   npm install
   ```
3. Start the server:
   ```bash
   cd server
   node app.js
   ```
4. Start the client:
   ```bash
   cd client
   npm start
   ```
